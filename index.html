Modular Kolonization System (MKS)

This is the repository for the MKS mod for Kerbal Space Program

NOTE: 0.18 is a BREAKING change for 0.17 users. Also, most docs on Github are still being updated to reflect the new changes - be patient :)

Required mods

  • TAC Life Support [YOU MUST USE THE 0.9 PRE RELEASE!]

STRONGLY recommended mods

(These are not technically required, but might as well be because without them, the mod is significantly less playable)

  • KAS (for linking up MKS modules and anchoring to the surface)
  • SCANsat (for finding resources)

Included Plugins

  • Module Manager
  • Open Resource System (alternate to Kethane)
  • TweakScale

Recommended mods

  • Cargo Transportation Solutions (great parts for transporting MKS modules)
  • Infernal Robotics (for assembling bases)
  • Procedural Fairings (for enclosing groups of MKS modules)
  • Ship Manifest (for getting around your base)
  • RealChute Parachute Systems (to avoid crashing a 20 ton station into Duna)

Compatible with

  • Kethane
  • Extraplanetary Launchpads (resource compatibility)
  • KSP Interstellar (reactors)
  • BioMass (resource compatibility)
  • BahamutoD's Drills (MM extension)
  • Connected Living Space (efficiency extensions)
